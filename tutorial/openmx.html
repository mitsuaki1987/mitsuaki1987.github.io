分子構造ファイルを入手する

ここでは題材としてメタン分子を扱う。 Googleで「Methane Mol-Instincts」で検索をする。

分子構造ファイルの形式変換

得られたSDF形式のファイルをOpenBabelというソフトウェアを使いXYZ形式に変換する。 

sudo apt install openbabel
macOSの場合はHomeBrew等を用いてインストールする。 

babel Methane.sdf Methane.xyz
